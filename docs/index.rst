Pathmaster Documentation
=======================

Pathmaster is a powerful command-line tool for managing your system's PATH environment variable, providing safe and efficient PATH manipulation with automatic backups and validation.

.. note::
   This project is being migrated from Rust to Go. See the Go Version section for details.
   The current Rust implementation remains fully functional.
